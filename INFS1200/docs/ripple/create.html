The creation task

Create a resource related to the course that your peers can use as a learning tool.

Creating your resource

  1. Pick the right category. You need to create a resource in the right category/ies to get marks. For more info, see here.
  2. Create your resource. For advice about this, see below.
  3. Submit your resource. You will see a button saying GET AI FEEDBACK. After clicking that, you should see the Submit button.

What counts as a "good" resource?

To judge if a resource is good, you should be aware of the moderation criteria:

  • Correctness and precision
    • All information in the resource is correct
    • If the resource has answer/s marked as "correct", the should be the actual correct answer/s
  • Quality of question (if the resource is a question)
    • Related to course content
    • Targets important concepts
    • No irrelevant details
    • Clearly written
  • Quality of options (if the question has options)
    • Used common mistakes/misconceptions
    • Used plausible options from the same category
    • Avoided "None of the above"
    • Avoided complex distractors (e.g. A and B)
  • Quality of explanation
    • Concise and accurate
    • Easy to understand
    • Used an encouraging tone

Note that a resource doesn't have to satisfy all of these criteria perfectly but it should at least have no major issues.

Give me my marks already!

If you have created the right number of resources before the due date, you will get the creation marks, even if the resource gets approved after the due date.

If you want to see your RiPPLE assessment progress, see here.

If you are worried about your resources being good enough to pass moderation, it is advised to create multiple resources to increase your chances.
