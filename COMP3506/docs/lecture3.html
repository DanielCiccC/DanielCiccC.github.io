Bucket sort

  • SSS be a list of nnn key, element items with keys in the range of [0, N-1]
  • Create a bucket for each element
  • Because the buckets are already sorted in the array, iterate through the buckets and link them together

Analysis

  • Initialising the bucket array has to take nnn time
  • Phase 1 takes O(n)O(n)O(n) time
  • Phase 2 takes O(n+N)O(n + N)O(n+N) time (has to iterate through each chain, in the whole bucket)
    • Which ever of nnn or NNN will dominate depening on the size.

Properties and Extensions

  • Stable sort property
  • Relative order of any two items with the same key is preserved after the execution of the algorithm
